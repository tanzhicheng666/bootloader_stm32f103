二、关键设计决策汇总

| 决策项            | 选择                          | 原因                        |
|-------------------|-------------------------------|-----------------------------|
| Flash 分区        | BL=0x08000000(16KB扇区), APP=0x08004000 | 扇区独立，互不损坏         |
| 参数区            | 0x08002000, 双镜像 256B×2      | 防掉电写入                  |
| 触发方式          | 500ms串口 + 按键5s + Flash标志  | 三重保障                    |
| 通信口            | USART1 (PA9/PA10), 115200      | 开发板调试口               |
| 时钟              | HSI 8MHz（不用外部晶振）        | 减少依赖，可靠              |
| 库依赖            | 零库依赖，直接寄存器操作        | 代码小，完全可控            |
| LED               | PC8                           | 你指定的                     |
| KEY1              | PB2                           | 你指定的                     |
